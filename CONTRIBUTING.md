# Contributing

Obrigado por visitar este repositório de estudos sobre observabilidade.

## Como contribuir

- Este projeto é focado em documentação e aprendizado.
- Se você quiser sugerir melhorias, abra uma issue no GitHub.
- Para enviar correções ou adições, faça um fork e crie um pull request.

## Padrões de contribuição

- Mantenha o conteúdo claro e objetivo.
- Use uma linguagem simples e profissional.
- Adicione referências quando usar exemplos ou definições externas.

## Aviso

Este repositório é principalmente uma atividade acadêmica. Contribuições são bem-vindas, mas não são obrigatórias.
